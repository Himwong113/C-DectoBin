# C-DectoBin
A demo code for Dec to Bin in C#

### **Concept:**
```
  1.Get a number num 
  2.mod num in 2 
  3.divde by 2
  4.Repeat 1-3 if !=0
  5.reverse the order
```
### **Example:**
```
num=10
10%2=0  -> arr=0    ||  num=10/2=5 
5%2=1   -> arr=01   ||  num=5/2=2 
2%2=0   -> arr=010  ||  num=2/2=1 
1=1     -> arr=0101

reverse 0101 -> 1010
Result =1010
```
